Projects

DevGlobe
Group Project using the MEAN Stack where I heavily worked in backend. DevGlobe uses D3.js to display past DevMountain students' "impact on the world" by mapping the locations of their jobs found after DevMountain on an interactive globe. In addition, we compared the costs of a typical 4 year University and other bootcamps to show how much DevMountain's courses can save students on average.
Beat
Beat is a web application created for a fictitious client to demonstrate my ability to create a modern blog layout. Using AngularJS and Less, this fully responsive web application also includes a functional archive page for users to search past blog posts. There are six main components of the website: Global Header, Articles, Archive, Popular Content, Newsletter Sign Up, and a Global Footer - All of which are Angular directives.
Divinity Salon
Static website created with AngularJS and customized for Divinity Salon located in Encinitas, California. This clean, simplistic website perfectly showcases Divinity Salon's minimalistic style and class. This design was created to be mobile and desktop friendly.
iSpend
Personal Project using the MEAN Stack. iSpend is a web application that allows users to keep track of their expenses at a company. In some industries it's hard to keep track of what employees spend out of pocket for their company - with iSpend, it's easy. (mobile only)
Appointment Scheduler
My very first full stack application using the MEAN Stack. Other technologies include JavaScript FullCalendar and D3.js. Application intended for a salon with a team of stylists to schedule their appointments and store past client information. Includes a "timecard" feature for employees' hours and an employee stats page that uses D3.js to display rankings of employees based on customer reviews.
